Address Standardization with NLP

Final Year Project - Asia Pacific University

Data Science focuses on showcasing Python knowledge and expertise with data science frameworks including pandas, scikit-learn, SciPy, seaborn, spaCy, Flask, TensorFlow, Keras, NumPy, and matplotlib.

This project addresses extraction of elements from unstructured textual data using NLP techniques.

Project Overview

Goal

E-commerce companies often process address data inefficiently, especially when standardizing addresses. This project aims to increase efficiency in address data standardization using BERT and CRF models for Named Entity Recognition (NER) and N-gram for the grammar correction model.

Result

Results are documented with properly conducted research, accuracy graphs, and final model outcomes. The model was deployed in a prototype website where users can enter addresses and receive automatically generated standardized results.

Duration

This type of project typically takes 2 weeks to 3 months, consisting of four phases:

  • Documentation - Creating materials for stakeholders and clients to understand the model
  • Modeling - Building and training the model (duration varies with complexity)
  • Data Preparation - Cleaning and preparing data for deployment
  • Deployment - Implementation based on client requirements

Data Understanding

Understanding the data structure allowed proper tagging of each word. The data was simplified into key elements.

Example address:
Jl.janaka Rt 1 Rw 1 krajan Wringinanom Sambit Kabupaten Ponorogo Jawa Timur

Elements identified:

  • Jl Janaka — Street name
  • RT 1, RW 1 — Neighborhood Unit and Community Unit
  • Krajan — Village
  • Wringinanom — District
  • Kabupaten Ponorogo — Regency
  • Jawa Timur — Province

This understanding enabled easier model building with clear visualization.
