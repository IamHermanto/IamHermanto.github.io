Data Science

Data Science focuses more on showcasing my python knowledge as well as knowledge on data science framework (pandas, scikit-learn, SciPy, seaborn, spaCy, flask, tensorflow, keras, NumPy, matplotlib).


The case below is my work from Asia Pacific University, Final Year Project regarding extraction of elements from unstructured textual data using NLP technique.

Goal

Often, companies process data inefficiently especially when it comes to e-commerce company when standardizing address data. The goal of this project is to increase the efficiency when standardizing address data using BERT and CRF model for the NER and N-gram for the grammar correction model.

Result

Results are presented in document where research are properly conducted and accuracy graphs are shown as well as the result of the final model. The model also was deployed in prototype website, where user are able to enter address and final result are automatically generated by the model.

Duration

This type of project varies between 2 weeks to 3 months. The project may contain 4 phases:

  • The first phase is the documentation, as it need to be made for stakeholders or client to be able to understand what's going on and how the model works.

  • The second phase is the modelling process, depending on the complexity of the model the time varies for the modelling process.

  • The third phase is the data preparation, the data need to be cleaned and readable for the model that will be deployed.

  • The fourth phase is the deployment, depending on the client request the deployment time needed may varies.

Data Understanding

Understanding the data allow me to properly tag each of the words thus in this process I simplified the data into few elements.

Example of the data:

Jl.janaka Rt 1 Rw 1 krajan Wringinanom Sambit Kabupaten Ponorogo Jawa Timur

  • Jl Janaka — street name
  • RT 1, RW 1 — Neighborhood Unit and Community Unit
  • Krajan — Village
  • Wringinanom — District
  • Kabupaten Ponorogo — Regency
  • Jawa Timur — Province
